Ключевые изменения по макету:
- Фото отображаются на прозрачном/тёмном фоне без рамок и контейнеров — вырезанные фигуры
- По умолчанию фото в ч/б (`grayscale`), при наведении становятся цветными (`grayscale-0`)
- Имя — белый, жирный шрифт (font-semibold)
- Должность — приглушённый белый текст
- Текст выровнен по центру под фото
- Убрать `aspect-[3/4]` и `bg-white/5` — фото без фона, просто img
- Добавить `transition-all duration-500` для плавного перехода ч/б → цвет

### Технические детали

Компонент будет использовать статический массив с импортированными изображениями как фоллбэк/оверрайд для фото, если данные из БД не содержат нужных фото. Но основная логика останется динамической — фото берутся из `member.photo_url`, а CSS-фильтр `grayscale` применяется ко всем.
