
# План редизайна страницы "О компании"

## Обзор
Полная переработка флагманской страницы компании в точном соответствии с референсным дизайном. Страница будет состоять из 10 основных секций с геометрическими шестиугольниками, горизонтальным таймлайном, каруселью команды на темном фоне и видео-отзывами клиентов.

---

## Секции страницы

### 1. Hero секция с шестиугольниками
- Хлебные крошки: кнопка назад + "ГЛАВНАЯ"
- Метка "О КОМПАНИИ" маленькими caps
- Большой serif заголовок: "Art Estate — команда, созданная профессионалами рынка недвижимости."
- Геометрические шестиугольники справа (серые, оранжевые, бирюзовые)
- Белый фон, широкий контейнер

### 2. О компании + фото команды
- Двухколоночный layout
- Слева: бирюзовая акцентная линия, текст о компании (8+ лет на рынке, работа в Москве, СПб, ОАЭ)
- Справа: большое фото команды (`hero-team.jpg`)

### 3. Услуги (8 карточек)
- Заголовок "Услуги" с горизонтальной линией справа
- Сетка 3 колонки с нумерованными карточками (01/, 02/... 08/)
- Каждая карточка: номер (бирюзовый), заголовок UPPERCASE, описание italic
- Данные берутся из базы (hook useServices)

### 4. История Art Estate (горизонтальный таймлайн)
- Заголовок "История Art Estate" (serif, center)
- Горизонтальная линия с бирюзовыми точками
- Годы над линией: 2016, 2017, 2018, 2019, 2020, 2023, 2024
- Точки на линии под каждым годом

### 5. Основатели компании
- Заголовок "Основатели компании" (serif, center)
- Две карточки с coral/red рамкой слева
- Имя (serif), подробное описание опыта
- Без круглых фото (как на референсе - только текстовые карточки)

### 6. "Как появилась идея?" блок
- Muted rose/coral фон
- Serif заголовок + текст о происхождении компании

### 7. Партнерство (темная секция)
- Темно-синий/navy фон
- Белый текст о партнерстве с застройщиками и банками
- Бирюзовая кнопка "СЕРТИФИКАТЫ"

### 8. Преимущества (3 колонки с иконками)
- Три карточки: "Внушительный опыт работы", "Амбициозность компании", "Непрерывное развитие"
- Каждая: иконка сверху, заголовок (serif), горизонтальная линия, описание (italic)
- CTA секция внизу: "Заинтересовались? Свяжитесь с нами." + кнопка "КОНТАКТЫ"

### 9. Команда экспертов (карусель на темном фоне)
- Темно-серый/черный фон
- Заголовок "Команда экспертов" с бирюзовой линией под ним
- Подзаголовок "Опытный переговорщик на вашей стороне!"
- Карусель с черно-белыми фото, которые становятся цветными при hover
- Стрелки навигации по бокам
- Данные из базы (hook useTeamMembers)

### 10. Видео-отзывы клиентов
- Заголовок "Слово нашим клиентам"
- Стрелки навигации сверху справа (минималистичные)
- Карточки с видео: фото, имя (serif), бирюзовая линия, описание, кнопка play
- Кнопка "ВСЕ ОТЗЫВЫ" внизу
- Данные из базы (hook useReviews)

## Технические детали

### Типографика
- Заголовки: Aeroport (font-serif)
- Текст: Montserrat
- Описания услуг: italic стиль

### Цвета
- Бирюзовый: `text-primary` / `bg-primary`
- Coral/Red рамка: `border-[#B4776E]` или `border-accent`
- Navy фон: `bg-navy` / `bg-[#1a2836]`
- Muted rose: `bg-[#C4A088]` или аналог

### Адаптивность
- Mobile: одна колонка, вертикальный таймлайн
- Tablet: 2 колонки где возможно
- Desktop: полный дизайн с 3 колонками

### Анимации
- Hover на карточках команды: grayscale-0
- Smooth scroll между секциями
- Карусель с плавными переходами
