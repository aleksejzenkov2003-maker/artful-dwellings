
# План доработки секций "Команда экспертов" и "Слово нашим клиентам"

## Анализ различий

### Секция "Команда экспертов"

| Элемент | Референс | Текущая реализация |
|---------|----------|-------------------|
| Заголовок | По центру | Слева |
| Подзаголовок | По центру, под заголовком | Слева, под заголовком |
| Бирюзовая линия | Под заголовком, по центру | Под заголовком, слева |
| Стрелки навигации | По бокам карусели (слева и справа) | Сверху справа в header |
| Фото | Крупные, занимают больше места, без скруглений | Меньшие, со скруглениями (rounded-lg) |
| Имя/Роль | Не отображаются на референсе | Отображаются под фото |

### Секция "Слово нашим клиентам"

| Элемент | Референс | Текущая реализация |
|---------|----------|-------------------|
| Заголовок | По центру | Слева |
| Стрелки навигации | Простые стрелки (← →) по краям, одиночные | Круглые кнопки сверху справа |
| Карточки отзывов | Видео-кадр с info overlay внутри | Видео сверху, текст снизу отдельно |
| Имя клиента | Внутри видео, крупный serif белый | Под видео, мелкий |
| Описание | Uppercase, внутри видео под именем | Обычный текст под карточкой |
| Кнопка Play | Бирюзовый контур, справа внизу видео | По центру, белый |
| Бирюзовая линия | Короткая, под именем внутри видео | Под именем, вне видео |

## Изменения в AboutTestimonials.tsx

### 1. Центрирование заголовка
- Заголовок "Слово нашим клиентам" - text-center

### 2. Новый дизайн стрелок
- Простые стрелки (ArrowLeft, ArrowRight) вместо ChevronLeft/Right
- Позиционирование: слева и справа от карусели
- Без круглых кнопок, минималистичный дизайн

### 3. Полная переработка карточек отзывов
Текущая структура:
```text
┌─────────────────┐
│   [Видео]       │
│   [Play btn]    │
├─────────────────┤
│ Имя             │
│ ───             │
│ Описание...     │
└─────────────────┘
```

Новая структура (как на референсе):
```text
┌─────────────────────────┐
│                   ┌───┐ │
│ Имя               │ ▶ │ │
│ ───               └───┘ │
│ ОПИСАНИЕ UPPERCASE      │
│      [Видео фон]        │
└─────────────────────────┘
```

### 4. Детали новой карточки
- Видео/фото как фон карточки (aspect-[16/10] или больше)
- Градиент overlay для читаемости текста
- Имя - крупный serif белый, позиционирован слева внизу
- Короткая бирюзовая линия под именем
- Описание - uppercase, белый, мелкий шрифт
- Кнопка Play - справа, бирюзовый контур круга с треугольником

## Технические детали

### Новая структура карточки отзыва:
```tsx
<div className="relative aspect-[4/3] overflow-hidden rounded-lg group">
  {/* Фото/видео как фон */}
  <img src={...} className="absolute inset-0 w-full h-full object-cover" />
  
  {/* Градиент overlay */}
  <div className="absolute inset-0 bg-gradient-to-t from-black/70 via-black/20 to-transparent" />
  
  {/* Контент внизу */}
  <div className="absolute bottom-0 left-0 right-0 p-6">
    <h3 className="text-2xl font-serif text-white mb-2">Имя</h3>
    <div className="w-8 h-0.5 bg-primary mb-2" />
    <p className="text-xs uppercase tracking-wider text-white/80">ОПИСАНИЕ</p>
  </div>
  
  {/* Play button справа */}
  <button className="absolute bottom-6 right-6 w-14 h-14 rounded-full border-2 border-primary flex items-center justify-center">
    <Play className="w-5 h-5 text-primary" />
  </button>
</div>
```

### Позиционирование стрелок команды:
```tsx
<div className="relative">
  {/* Стрелка слева */}
  <button className="absolute left-0 top-1/2 -translate-y-1/2 -translate-x-4 z-10">
    <ChevronLeft />
  </button>
  
  {/* Карусель */}
  <div ref={emblaRef}>...</div>
  
  {/* Стрелка справа */}
  <button className="absolute right-0 top-1/2 -translate-y-1/2 translate-x-4 z-10">
    <ChevronRight />
  </button>
</div>
```
